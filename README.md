# Personal Expense Tracker

## Description

A CLI Personal Expense Tracker built using Python.

This program allows users to pick any given option:

* Add expenses
* View all expenses
* Filter expenses by category
* Store data in a CSV file

## What it does:

### Add Expense

Users can enter:

* Amount
* Category
* Note

The expense is saved to a CSV file and it gets appended.

### View Expenses

Choosing this displays:

* Date
* Category
* Amount
* Note

Also calculates total expenses.

### Filter By Category

Allows users to view expenses from a specific category and displays the subtotal.

## Categories

* Food
* Transport
* Entertainment
* Other

## How to Run
Three simple steps:
1. Open terminal inside the project folder
2. Run:
   python expense_tracker.py
3. Enter your data
